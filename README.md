# Simon Game

A web-based Simon Game built using **HTML, CSS, and JavaScript**. Test your memory by repeating the color patterns in the correct sequence. The project focuses on interactive UI and animations.

## Features

- Repeat color patterns to test memory
- Interactive animations and sound effects
- Responsive design for desktop and mobile
- CSS-based styling for visual effects

## Technologies Used

- HTML
- CSS
- JavaScript
